# Muslim Website Backend (Demo)
## Quick start
1. Copy `.env.example` to `.env` and set values.
2. Install dependencies: ``
3. Start server: `` (needs nodemon) or ``
4. Open http://localhost:5000

This project includes:

- MongoDB (mongoose)
- Simple classic-styled frontend pages in /public (signup, login, profile)

Keep your JWT secret safe in production.
